Landing Page: Projeto Aprenser — Página Web (HTML & CSS):Desenvolvimento de uma página web estática utilizando apenas HTML5 e CSS3, criada durante o início da minha jornada na programação. O projeto foi pensado para uso em desktop, sem responsividade, com foco total em praticar fundamentos essenciais, como: Estruturação clara e semântica do HTML, Organização de estilos em CSS, Construção de layout utilizando conceitos básicos da época, Separação adequada entre conteúdo e apresentação.
