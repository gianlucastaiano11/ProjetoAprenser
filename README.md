
Descrição:
Projeto desenvolvido no início da jornada em desenvolvimento web, com foco total na compreensão dos fundamentos de HTML e CSS. Trata-se de uma página web estática criada para desktop, priorizando a prática de estruturação e organização do código.

Objetivo do Projeto:
Consolidar os conceitos iniciais de desenvolvimento web
Praticar a estruturação clara do HTML
Entender a organização básica de estilos em CSS

Aprendizados:
Estruturação correta do HTML
Organização de arquivos CSS
Compreensão da separação entre conteúdo e apresentação
Base para evolução em projetos mais complexos e responsivos

Tecnologias Utilizadas:
HTML5
CSS3
